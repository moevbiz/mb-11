## Local development

### Prerequisites

- [Node and NPM](https://nodejs.org/)


## Running locally

```bash
# install the dependencies
npm install

# External data sources can be stashed locally for API-less dev
npm run seed

# It will then be available locally for building with:
npm run start
```
